Mở đầu về CSS

CSS là gì?

CSS (Cascading Style Sheets) là một ngôn ngữ dùng để mô tả cách trình bày và hiển thị của các phần tử HTML trên trang web.

Tại sao cần dùng CSS?

  • Tách biệt nội dung và kiểu dáng: Dễ dàng bảo trì và cập nhật trang web mà không làm thay đổi nội dung.
  • Kiểm soát phong cách: CSS cho phép quản lý màu sắc, font chữ, kích thước, khoảng cách giữa các phần tử, và bố cục trang web.
  • Tính nhất quán: Đảm bảo phong cách giống nhau trên tất cả các trang thông qua một tệp CSS duy nhất.
  • Thiết kế đáp ứng: Tự động điều chỉnh bố cục cho các thiết bị khác nhau, cải thiện trải nghiệm người dùng.
