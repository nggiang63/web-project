Kỹ thuật nhúng CSS vào HTML

Inline CSS

Nhúng CSS trực tiếp trong thẻ HTML: sử dụng thuộc tính style

Ví dụ:

  • Ưu điểm: Dễ dàng áp dụng và thay đổi cho từng phần tử riêng lẻ.
  • Nhược điểm: Không tái sử dụng được và làm mã HTML trở nên khó đọc.

Internal CSS

Nhúng CSS vào phần <head> của tài liệu HTML: sử dụng thẻ <style>

Ví dụ:

  • Ưu điểm: Dễ quản lý và tái sử dụng cho toàn bộ tài liệu.
  • Nhược điểm: Không thể sử dụng cho nhiều tài liệu HTML khác nhau.

External CSS

Sử dụng một tệp CSS riêng biệt được liên kết với tài liệu HTML thông qua thẻ <link>.

Ví dụ:

  • Ưu điểm: Tái sử dụng dễ dàng và giúp tách biệt nội dung và kiểu dáng.
  • Nhược điểm: Cần phải quản lý nhiều tệp.
