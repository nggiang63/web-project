CSS là gì?

1. Định nghĩa CSS

Cascading Style Sheets (CSS) là một ngôn ngữ được sử dụng để mô tả cách trình bày của một tài liệu viết bằng HTML hoặc XML.

2. Tại sao cần CSS?

CSS giúp tách biệt nội dung và kiểu dáng, cho phép dễ dàng duy trì và thay đổi thiết kế của trang web.

3. Cách hoạt động của CSS

CSS hoạt động bằng cách chọn các phần tử trong HTML và áp dụng các quy tắc kiểu dáng cho chúng.
